## What is SportsMailPro?

SportsMailPro is a self-hosted email marketing platform built specifically for teams, leagues, venues, and sports organizations. Send campaigns, manage subscriber lists, track engagement — without monthly SaaS fees.

## Features

- 📋 **Multiple email lists** with segmentation for fans, sponsors, media, staff
- 📧 **Send emails** via SMTP credentials or AWS SES
- 🔗 **Subscribe endpoint** — drop a form on any website
- 📥 **Import subscribers** from CSV or paste
- ✅ **Double opt-in** and one-click unsubscribe (GDPR/CAN-SPAM ready)
- 🎨 **HTML campaign builder** — send any design you want
- 📊 **Analytics** — open rates, click rates per campaign and per subscriber
- 🔗 **Link tracking** — see which links get clicked
- 🤖 **AI-ready** — integrates with your SIR AI assistant

## Sports-Specific Templates Included

- 🏈 Game Day Announcements
- 🎟️ Ticket Sale Campaigns
- 📰 Weekly Team Recaps
- 🤝 Sponsor Update Newsletters
- 🏆 Season Highlights
- 📣 Fan Engagement Surveys

## Quick Start

```bash
git clone https://github.com/SIR-Sports-Tech/SportsMailPro.git
cd SportsMailPro
npm install
cp .env.dist .env.local
# Edit .env.local with your MongoDB + SMTP/SES credentials
npm run dev
```

## Tech Stack

- **Next.js 14** — React framework
- **MongoDB** — Database
- **Nodemailer** — Email delivery (SMTP or AWS SES)
- **TypeScript** — Type-safe codebase
- **Tailwind CSS** — Styling
